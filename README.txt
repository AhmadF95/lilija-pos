# Mini Store Desktop (Electron)

هذا مشروع جاهز لتحويل تطبيق المتصفح إلى برنامج سطح مكتب يمكن تنصيبه.

## المتطلبات
- Node.js LTS (يفضّل 18+)
- أول مرة فقط: اتصال إنترنت لتحميل حزم npm

## تشغيل مباشر بدون تنصيب
```
npm install
npm start
```
سيفتح البرنامج نافذة التطبيق مباشرة.

## إنشاء مُثبّت (Windows)
```
npm install
npm run build:win
```
ستجد ملف مثبت `.exe` وممكن ملف `Portable.exe` داخل مجلد `dist/`.

## ملاحظات
- البيانات تُخزّن محليًا داخل مساحة تطبيق Electron (localStorage) لكل مستخدم على جهازه.
- لا يحتاج البرنامج لأي خادم خارجي — كل شيء يعمل أوفلاين.
- إذا أردت أيقونة مخصّصة، أنشئ مجلد `build/` وضع الأيقونات وفق electron-builder.
